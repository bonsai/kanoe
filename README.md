# 長良川カヌー川下りゲーム 🚣

MediaPipe ポーズ検出を使った顔認識カヌーゲーム

## 🎮 操作方法

### 顔認識操作（推奨）
- **👄 口をパクパク** → 漕ぐ（加速）
- **👇 鼻を下に動かす** → しゃがむ（橋を通過）
- **👈 頭を左に傾ける** → 左に移動
- **👉 頭を右に傾ける** → 右に移動

### キーボード操作
- **↑キー / W** → 漕ぐ（加速）
- **↓キー / S** → しゃがむ
- **←キー / A** → 左に移動
- **→キー / D** → 右に移動

## 🚀 起動方法

### 方法1: ローカルサーバー（推奨）

#### Windows
```bash
# PowerShell
.\START_SERVER.ps1

# または コマンドプロンプト
START_SERVER.bat
```

#### Python がある場合
```bash
python -m http.server 8000
```

#### Node.js がある場合
```bash
npx http-server -p 8000
```

ブラウザで `http://localhost:8000` を開く

### 方法2: オンライン版
デプロイ済みの URL にアクセス（Vercel など）

## 🐛 デバッグモード

### 有効化方法

1. **ボタンで切り替え**
   - 左上の「🐛 DEBUG」ボタンをクリック

2. **URL パラメータ**
   ```
   http://localhost:8000/?debug=true
   ```

### デバッグパネルの機能
- リアルタイムのコンソールログ表示
- エラー、警告、成功メッセージの色分け
- タイムスタンプ付きログ
- 最新50件のログを保持

## 📋 必要要件

- モダンブラウザ（Chrome, Edge, Firefox など）
- Webカメラ
- HTTPS または localhost での実行（カメラアクセスのため）

## 🎯 ゲームの目的

長良川を下りながら、10個の橋を通過して海を目指します。
橋が近づいたらしゃがんで通過しましょう！

## 🔧 トラブルシューティング

### カメラが起動しない
- ブラウザのカメラ権限を確認
- HTTPS または localhost で実行しているか確認

### 橋が表示されない
- ローカルサーバーで実行しているか確認
- bridges.json が同じディレクトリにあるか確認
- デバッグモードでエラーログを確認

### ポーズ検出が動作しない
- インターネット接続を確認（MediaPipe CDN）
- カメラに顔が映っているか確認
- 右下の小さなカメラ画面で緑色の線が表示されるか確認

### ゲームが重い
- pose-controller.js の modelComplexity を 0 に変更
- script.js の波の解像度を下げる

詳細は `DEBUG_GUIDE.md` を参照してください。
